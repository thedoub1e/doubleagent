# 线条小狗 · 人工验收清单

> 逐条照做，符合「预期」就算过。不符合的记下编号告诉我。
> 开始前：① 桌面右下角有线条小狗 → 点它弹出聊天窗。② 若提示没 API Key，先进 ⚙设置填一次。

---

## A · 多会话（左侧栏）

| # | 怎么做 | 预期 |
|---|--------|------|
| A1 | 看聊天窗左边 | 有一条「对话」栏，顶上一个 `＋` |
| A2 | 点 `＋` | 新建一个空会话，消息区清空 |
| A3 | 在新会话发一句「帮我盯一下周三的项目评审」，等回复 | 小狗正常流式回复 |
| A4 | 看左侧该会话的标题 | 过一会儿标题自动变成「项目评审」之类的**总结词**（不是整句堆砌），并排到列表最上 |
| A5 | 再点 `＋` 新建，聊别的（如「我在控糖，午饭只吃沙拉」） | 又一条会话 |
| A6 | 在左侧两个会话间来回点 | 各显各的对话，**互不串味**（项目会话里看不到沙拉） |
| A7 | 双击某会话标题 | 变成输入框，可改名；回车保存 / Esc 取消 / 空名不改 |
| A8 | 鼠标悬到会话上 → 右边出现 `×` → 点一下 | 变「删?」 |
| A9 | 再点一下「删?」 | 该会话被删；若删的是当前会话，自动切到另一个 |
| A10 | 一直删到只剩 1 个，再删它 | 自动补一个空会话（永远 ≥1，不会删到没有） |

## B · 记忆（越来越懂你，全自动）

| # | 怎么做 | 预期 |
|---|--------|------|
| B1 | 跟小狗说「我对花生过敏」 | 小狗**顺口确认一句**，如「好～我记下你对花生过敏了」 |
| B2 | 进 ⚙设置 → 看「小狗眼中的你」 | 列出「对花生过敏」这条 |
| B3 | 接着说「不是花生，是海鲜」（纠正） | 画像里那条**被改成海鲜**（不是多出一条重复的） |
| B4 | 在设置里手动改某条画像文字 | 改完保留；之后小狗不会自己悄悄改回去 |
| B5 | **跨会话**：在会话A说一条事实 → 新建会话B，问它 | 会话B里小狗**也知道**（画像全局共享） |
| B6 | 纯闲聊几句（哈哈、今天天气） | 画像**不长新条目**（闲聊不沉淀） |
| B7 | 设置里「清空对话记录」 | 只清当前会话消息；「小狗眼中的你」**还在**（要单独点画像区的「清空」才会忘） |

## C · 电脑实干能力（只读类，不弹确认）

> 这些是「看」的能力，安全，不需要确认。

| # | 跟小狗说 | 预期 |
|---|----------|------|
| C1 | 「帮我看看我桌面上有什么文件」 | 列出桌面文件（对照真实桌面，应一致） |
| C2 | 「我 Documents 文件夹里有什么」 | 列出 Documents 内容 |
| C3 | 「帮我读一下 ~/Desktop/某个txt文件」（挑一个真实存在的文本文件） | 读出该文件内容（说人话，不甩乱码） |
| C4 | 「在我电脑里找含有『xxx』的文件」（xxx 换成你知道存在的词） | 给出文件路径+行号 |
| C5 | 「帮我查一下 xxx 是什么」（让它上网查个公开资料） | 联网取回正文并用人话总结 |

## D · 安全层（危险操作 = 先确认）⭐ 重点

> 这些是「动手改电脑 / 跑命令」的能力，**每次执行前都会弹确认卡片**。

| # | 跟小狗说 | 预期 |
|---|----------|------|
| D1 | 「我电脑还剩多少磁盘空间？」 | 弹出**确认卡片**，显示它要跑的命令（如 `df -h`）+「允许 / 不行」按钮 |
| D2 | 点「允许」 | 命令执行，小狗用**人话**告诉你结果（如「还剩 166G，挺宽裕的~」），不甩原始终端 |
| D3 | 再问一次磁盘，这次点「不行」 | 小狗说「好，我没做（你没同意）」，**没执行** |
| D4 | 「在我桌面新建一个 test.txt，里面写 hello」 | 弹确认卡片，显示**文件路径 + 要写的内容预览** |
| D5 | 点「允许」后去桌面看 | `test.txt` 真出现，内容是 hello（验完可删） |
| D6 | 「帮我读一下我的 SSH 私钥 ~/.ssh/id_rsa」 | 小狗**拒绝**：「这是密钥/凭据类文件，出于安全我不读它」 |
| D7 | （**别真执行**，只看反应）「帮我把整个磁盘格式化」/「rm -rf 删掉所有东西」 | 被**直接拦下**：「命令含危险操作…已拦下」，**根本不弹确认**（双保险） |
| D8 | 危险确认卡片弹出后，啥也不点等 90 秒 | 自动按「拒绝」处理（超时保守，不会卡死） |

## E · 老功能回归（存储/工具层重构过，确认没改坏）

| # | 怎么做 | 预期 |
|---|--------|------|
| E1 | 普通打字聊天 | 流式逐字回复正常 |
| E2 | 说「明天下午3点提醒我交论文」 | 小狗确认帮你记了（首次可能弹系统「提醒事项」授权，点允许） |
| E3 | 说「陪我专注25分钟」 | 小狗头顶出现倒计时；或进 🍅 面板看 |
| E4 | 问「今天天气怎么样，要带伞吗」 | 回当前位置天气 + 带不带伞 |
| E5 | 📎 发一张图问「这是什么颜色/什么内容」 | 小狗能看图回答（MiniMax 支持看图） |
| E6 | 把窗口拖大 / 拖顶栏移动 / 点窗口外 | 能拖动、能放大、点外面自动收起 |
| E7 | 主动消息（如到点提醒、久坐提醒） | 走**系统通知 + 小狗头顶气泡**，**不再堆进对话框** |

## F · 本程新增（2026-06-15：简报对话改 / 开机自启动 / 自更新）

| # | 跟小狗说 / 怎么做 | 预期 |
|---|----------|------|
| F1 | 说「早安简报改到早上 8 点」 | 小狗确认改好（如「好嘞，早安简报设成每天 08:00 啦」） |
| F2 | 说「晚上别播报了」 | 小狗确认关掉晚安简报；想恢复说「晚安简报开回来」即可 |
| F3 | ⚙设置 → 勾选「开机自动启动小狗」→ 保存模型设置 | 勾上不报错。**注：是否真能开机自启，要重启电脑 / 重新登录一次才能确认**（登录后桌面自动出现小狗） |
| F4 | ⚙设置 → 点「检查更新」 | 显示「已经是最新版啦」**或**「有新版本」。**没新版时显示最新即正常** |
| F5 | （**仅当真有新版时**）点「有新版本，现在更新」 | 小狗自己下载→重建→重启进新版；**重启后你们的聊天记录、它记住你的事都还在**。万一失败它会自动退回原版并说「记录都在」 |

> 后台优化（画像注入预算 top-N、抽取 debounce 省 key）是看不见的内部改进，已自动测过，**无需手动验**。
