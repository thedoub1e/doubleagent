# 🐶 线条小狗 · 桌面 AI 陪伴

一只常驻在 macOS 桌面上的线条小狗，点一下就能像 Gemini 那样聊天。
它会**陪伴**你、温柔**监督**你的学习作息、随时**聊天**和**解惑** —— 给在外求学的你一点陪伴。

> 仅 macOS。模型用国产 **MiniMax**（默认），也支持 DeepSeek / 智谱GLM / Kimi / 通义 / Gemini 反代。
> API Key 只存在你自己电脑本地，**绝不上传、绝不进 git**。

---

## 一、安装（三步，不用碰命令行）

1. **拿到代码**：用 **GitHub Desktop**（推荐，最省心）把这个仓库 clone 到本地；
   或在仓库页点绿色 `Code` → `Download ZIP` 下载后解压。
2. **双击 `安装.command`**：
   - 如果电脑没装 Node.js，它会自动打开下载页，下载并安装 **LTS 版本**（一路点继续），
     装好后**再双击一次**`安装.command`。
   - 看到 `✅ 安装完成` 就好了。
3. **双击 `启动小狗.command`**：桌面右下角就会出现小狗 🐶。

> 💡 第一次双击 `.command` 如果提示「无法验证开发者 / 已阻止」，
> **右键点它 → 选「打开」→ 再点「打开」** 即可（只需这一次）。

---

## 二、填 API Key（第一次用）

小狗要连模型才能说话，需要一个 **MiniMax** 的 Key：

1. 打开 [platform.minimaxi.com](https://platform.minimaxi.com)（**国内站**）注册登录。
2. 在「账户管理 / 接口密钥」里**创建一个 API Key**，复制下来。
3. 在小狗聊天窗右上角点 **⚙ 设置** → 把 Key 粘进 **API Key** 框 → **保存**。

填好后，点小狗、打字、回车，它就会回复你啦。

> 默认模型源是 **MiniMax · 国内**、模型 **MiniMax-M3**。在国外如果连不上，
> 可在设置里换成 **MiniMax · 国际**（需用国际站 minimax.io 的 Key）。

---

## 三、怎么玩

- **点小狗** → 打开 / 收起聊天窗
- **按住小狗拖** → 把它挪到喜欢的位置
- **聊天窗 ⚙ 设置** 里可以：
  - 换**模型源 / 模型**
  - 开关**主动监督**、改提醒时间和内容（默认每晚 21:00 问学习、23:30 催早睡）
  - **换形象**：选一张你喜欢的图片 / GIF 当小狗（支持线条小狗素材）

## 四、常见问题（FAQ）

**Q：双击 `.command` 弹「无法验证开发者」？**
A：右键点它 → 「打开」→ 「打开」。只需第一次这样。

**Q：安装卡在下载、很久不动？**
A：多半是网络。安装脚本已自动用国内镜像；连好网后再双击一次 `安装.command`。

**Q：聊天报红色「401 / invalid api key」？**
A：Key 站点不对。**国内站(minimaxi.com)的 Key** 要配 **MiniMax · 国内**；
国际站(minimax.io)的 Key 配 **MiniMax · 国际**。在设置里对应切换。

**Q：小狗挡住东西了？**
A：透明区域是可以点穿的；按住小狗本体拖动就能挪走它。

**Q：怎么让它别一直开着？**
A：关掉那个黑色终端窗口，小狗就下班了；想它回来再双击 `启动小狗.command`。

## 隐私与安全

- API Key 只写在本地（应用数据目录 / 项目 `.env`），**不进 git、不上传任何服务器**。
- 没有云端后端、没有账号系统、没有遥测：你的聊天只在你电脑和你选的模型厂商之间。
