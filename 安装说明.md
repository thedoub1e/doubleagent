# 🐶 线条小狗 · 安装说明（macOS）

给你的桌面小狗，陪你的小宝贝～ 按下面三步装好就行。**全程只需复制粘贴。**

---

## 第一次安装（只做一次）

### 第 1 步：装 Node.js（小狗运行需要的底座）

浏览器打开 👉 **https://nodejs.org/zh-cn**
点绿色的 **LTS** 按钮下载，下到一个 `.pkg` 文件，**双击它**，一路点「继续 / 安装」装好。

### 第 2 步：打开「终端」

按 `Command(⌘) + 空格`，输入 `终端` 或 `Terminal`，回车打开那个黑色窗口。
> 终端打开时，默认就停在你的「个人主目录」（也就是你的「家」📂）。小狗就会装在这里面，方便又好找。

### 第 3 步：在终端里，一行一行粘贴下面的命令

每粘贴**一行**就按一次回车，等它跑完（光标重新闪烁）再贴下一行：

**①** 确认停在你的「家」目录：
```bash
cd ~
```

**②** 把小狗下载下来（会自动新建一个叫 `doubleagent` 的文件夹）：
```bash
git clone https://github.com/thedoub1e/doubleagent.git
```
> 如果这步弹窗让你装「命令行工具」，点【安装】，装完再贴一次这行就好。

**③** 进入小狗的文件夹：
```bash
cd doubleagent
```

**④** 开始安装（第一次会跑几分钟，泡杯茶☕）：
```bash
bash scripts/install.command
```

跑完之后，**小狗就出现在桌面右下角啦** 🎉
以后它就住在 `你的用户名 / doubleagent` 这个文件夹里。

> 💡 嫌麻烦？也可以把上面 ①~④ 合成一整行一次粘贴：
> ```bash
> cd ~ && git clone https://github.com/thedoub1e/doubleagent.git && cd doubleagent && bash scripts/install.command
> ```

> 装好后建议点开小狗 →「设置」→ 勾上 **「开机自动启动」**，这样每次开机它都会自己出现来陪你。

---

## 怎么更新（有新功能 / 修了 bug 时）

**最省事**：点小狗 →「设置」→ 点 **「检查更新」**，有新版就点「现在更新」，它会自己下载、重启。

> 💚 **放心**：更新只换小狗的"程序"，**你和它的聊天记录、它记住你的所有事，都不会丢、不会被覆盖**。
> （这些都存在系统的应用数据目录里，跟程序是分开的；万一新版改了存储格式，它会先自动备份再升级。）

如果设置里的「检查更新」点不动，也可以在终端跑（备用）：

```bash
bash ~/doubleagent/scripts/update.command
```

---

## 遇到问题？

- **小狗没出现**：在终端跑 `bash ~/doubleagent/scripts/install.command` 再试一次。
- **想手动启动一下**：终端跑 `cd ~/doubleagent && ./node_modules/.bin/electron-vite preview`。
- 实在不行，把终端里红色的报错截图发给 **送你小狗的那个人** 看看就好～ 🐾
