# 🐶 线条小狗 · 安装说明（macOS）

给你的桌面小狗，陪你的小宝贝～

---

## ✨ 最简单：一条命令，全自动装好

**第 1 步：打开「终端」**
按 `Command(⌘) + 空格`，输入 `终端` 或 `Terminal`，回车，打开那个黑色窗口。

**第 2 步：把下面这一整行复制进去，按回车**

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/thedoub1e/doubleagent/main/scripts/setup.sh)"
```

然后**跟着提示走**就行。中途可能会：
- 让你**输一次开机密码**（输的时候屏幕不显示，是正常的，打完按回车）
- 让你**按一次回车**确认

它会自动帮你：装好需要的小工具（Homebrew / Node）→ 下载小狗 → 构建 → 启动。

几分钟后（第一次稍慢，泡杯茶☕），**小狗就出现在桌面右下角啦** 🎉
> 小狗会自动装在「你的用户文件夹 / `doubleagent`」里——**不用你自己建文件夹**，它会建好。

---

## 装好之后

1. **点一下小狗**，打开聊天窗。
2. 进「**设置**」→ 选一个「模型源」→ **粘贴一个 API Key** →「保存」。（如果送你小狗的人已经替你填好了，就跳过这步。）
3. 在「设置」里勾上「**开机自动启动**」，以后每次开机它都自己出现来陪你。

就可以开始跟它聊天啦～ 🐾

---

## 怎么更新（有新功能 / 修了 bug 时）

点小狗 →「设置」→ 点 **「检查更新」**，有新版就点「现在更新」，它会自己下载、重启。

> 💚 **放心**：更新只换小狗的"程序"，**你和它的聊天记录、它记住你的所有事，都不会丢、不会被覆盖。**

---

## 如果那条命令中途卡住了（备选办法）

极少数情况下自动装会卡（比如网络问题）。可以分开来一步步跑：

```bash
# 1) 装 Homebrew（Mac 软件管家，已装会自动跳过）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2) 装 Node（小狗的运行底座）
brew install node

# 3) 下载小狗到「用户文件夹/doubleagent」并安装启动
git clone https://github.com/thedoub1e/doubleagent.git ~/doubleagent && cd ~/doubleagent && bash scripts/setup.sh
```

实在不行，把终端里红色的报错截图发给 **送你小狗的那个人** 看看就好～ 🐾
