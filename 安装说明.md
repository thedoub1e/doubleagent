# 🐶 线条小狗 · 安装说明（macOS）

给你的桌面小狗，陪你的小宝贝～ 全程只需复制粘贴。

---

## 第 1 步：给小狗找个家

在你喜欢的位置（桌面、文稿都行）**新建一个文件夹**，名字随意（比如 `doubleagent`）。
小狗就会装进这个文件夹里。

## 第 2 步：在这个文件夹里打开「终端」

任选一种：

- **方法 A（最稳）**：打开「终端」App（`⌘+空格` 搜 `终端` / `Terminal`），
  输入 `cd ` （**cd 后面带一个空格**），然后把刚建的文件夹**拖进终端窗口**，按回车。
- **方法 B**：在「访达」里右键那个文件夹 →「服务」→「**在文件夹位置打开"终端"**」。
  （如果右键里没有这一项，用方法 A。）

## 第 3 步：粘贴这一行，回车

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/thedoub1e/doubleagent/main/scripts/setup.sh)"
```

然后**跟着提示走**。中途可能会：
- 让你**输一次开机密码**（输的时候屏幕不显示字，是正常的，打完按回车）
- 让你**按一次回车**确认

它会自动帮你：装好需要的小工具（Homebrew / Node）→ 把小狗下载进你这个文件夹 → 构建 → 启动。

几分钟后（第一次稍慢，泡杯茶☕），**小狗就出现在桌面右下角啦** 🎉

---

## 装好之后

1. **点一下小狗**，打开聊天窗。
2. 进「**设置**」→ 选一个「模型源」→ **粘贴一个 API Key** →「保存」。（送你小狗的人若已替你填好，就跳过。）
3. 勾上「**开机自动启动**」，以后每次开机它都自己出现来陪你。

开始聊天吧～ 🐾

---

## 怎么更新

点小狗 →「设置」→ 点 **「检查更新」**，有新版就点「现在更新」，它自己下载、重启。

> 💚 **放心**：更新只换小狗的"程序"，**你和它的聊天记录、它记住你的所有事，都不会丢、不会被覆盖。**

> 备用：也可以进小狗文件夹，双击 `scripts` 里的 `update.command`。

---

## 遇到问题

如果第 3 步那行中途卡住（多半是网络），可以分开一步步跑（在同一个文件夹的终端里）：

```bash
# 1) 装 Homebrew（已装会自动跳过）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
# 2) 装 Node
brew install node
# 3) 把小狗下载进当前文件夹并安装启动
git clone https://github.com/thedoub1e/doubleagent.git . && bash scripts/setup.sh
```

实在不行，把终端里红色的报错截图发给 **送你小狗的那个人** 看看就好～ 🐾
